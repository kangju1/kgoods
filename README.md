## 프로젝트 핵심 한 줄 요약

> 웬즈데이가 수년간 구축한 K-콘텐츠 소싱 네트워크를 디지털 B2B 플랫폼으로 전환하여, 현재 수작업으로 처리되는 신보 안내 → 발주 → 출고 → 정산 전 사이클을 자동화하고, 글로벌 바이어가 직접 주문할 수 있는 wholesale 플랫폼을 구축한다.

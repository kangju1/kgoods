# 벤치마크 분석

## 1. StyleKorean Wholesale

**URL**: https://stylekoreankbeautywholesale.com/

### 플랫폼 특징
- K-뷰티(화장품) 전문 B2B 도매 플랫폼
- 브랜드 알파벳 인덱스 + 카테고리(스킨케어, 메이크업, 헤어 등) 이중 탐색
- 로그인 전: 상품 노출 O / 가격 비공개
- 로그인 후: 가격 및 장바구니 기능 활성화
- 바코드(EAN) 기반 상품 식별
- CPNP / SCN 등 국제 인증 마크 표시 → 해외 유통 적합성 명시
- Best Sellers / New Arrival / Promotion / Clearance 섹션 분리

### Koreal에 적용 가능한 인사이트
- **로그인 기반 가격 공개**: 거래처별 공급가 차등화 구현 시 동일 방식 적용
- **바코드(JAN/EAN) 기반 상품 식별**: K-POP 상품도 JAN 코드 활용 가능
- **카테고리 구조**: 아티스트 / 상품 유형(음반·잡지·굿즈) 이중 분류 체계 참고
- **신규입고 / 베스트셀러 구분**: 신보와 구보를 명확히 분리

## 2. Ktown4u

**URL**: https://kr.ktown4u.com/

### 플랫폼 특징
- K-POP 음반·굿즈 전문 커머스 (B2C이나 구조 참고 가능)
- 아티스트별 상품 그룹핑, 공식 응원봉 별도 섹션
- 예약(Pre-Order) 주문 → 차트 집계 연동 (하이토 차트, 써클 차트)
- 브랜드샵 구조 (팬덤 단위 상품 큐레이션)
- 일본 시부야109 협력 등 글로벌 연계

### Koreal에 적용 가능한 인사이트
- **아티스트 중심 상품 구조**: B2B에서도 바이어가 아티스트별로 상품 탐색하는 UX 필요
- **예약(Pre-Order) 주문**: 신보 예약판매 → 출시 후 출고 구조 그대로 적용
- **세트 vs 랜덤 옵션**: 음반 특성(랜덤 1종, SET 불가 등 유통사 조건) 상품 속성으로 관리
- **다국어 지원**: 글로벌 바이어 타겟이면 영어·일본어 지원 필수

## 3. Nippan IPS (ts-base.jp)

**URL**: https://www.ts-base.jp/logi/ips-export/

### 플랫폼 특징
- 일본 최대 도서·잡지·굿즈 유통사 Nippan(일본출판판매주식회사) 해외사업 법인의 도매 플랫폼
- 재고 기반 판매 (예약판매는 캐릭터 굿즈 일부만)
- 전 세계 거래처를 대상으로 주문 접수
- 상품 탐색: 商品検索 → 신착순(新着順) 정렬 가능
- 계정 공유 엄격 제한, 주문 후 취소 어려움

### Koreal에 적용 가능한 인사이트
- **비즈니스 기회**: 해당 플랫폼 상품·재고 크롤링 연동 시 Nippan의 해외 재고 판매 채널 확보 가능
- **운영 참고**: 재고 기반 즉시 주문 + 예약 주문 두 가지 모드 운영 모델 참고
- **글로벌 B2B 플랫폼 표준**: 계정 등록 심사 → 가격 공개 → 주문 프로세스가 국제 B2B 표준에 가까움

## 4. 경쟁 우위 포인트

Koreal이 위 벤치마크 대비 차별화할 수 있는 영역:

1. **신보 자동화**: 공급사 이메일 수신 → 자동 파싱 → 플랫폼 등록 → 바이어 알림까지 자동화. 현재 어떤 경쟁사도 이 과정을 완전 자동화하지 않음
2. **K-POP 특화 상품 속성 관리**: 랜덤/세트 판매 조건, 아티스트별 유통 제한, 단독 유통 상품 등 K-POP 도매에 특화된 속성 체계
3. **이종 발주 채널 통합**: 이메일·카카오톡·엑셀 등 분산된 발주 채널을 단일 플랫폼으로 통합
4. **일본 상품 현지 소싱 강점**: Nippan IPS와의 직거래 네트워크 보유, 일본 발행 잡지·화보집 등 국내 다른 플랫폼에서 취급하지 않는 상품 카테고리
